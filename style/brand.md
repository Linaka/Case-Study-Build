# Brand style reference

This local reference is derived from the supplied `Uber.md` design-system notes.

## Direction

- Monochrome, restrained, editorial.
- Black and white are the lead surfaces; soft greys are supporting surfaces only.
- No decorative accent colour.
- Sentence-case display headlines.
- Large confident typography.
- Pill metadata chips use `999px` radius.
- Cards and large content surfaces use `16px` radius.
- Image frames use `4:3` or `16:9` ratios.
- Case-study PDFs alternate between white pages and black feature-band pages.
- Depth is mostly flat, with only subtle shadow where a panel needs separation.

## Colour roles

- Ink black: `#000000` for dark bands, text, and primary actions.
- Canvas: `#ffffff` for default pages.
- Canvas soft: `#efefef` for chips and quiet panels.
- Canvas softer: `#f6f6f6` for nested form fields.
- Body: `#5e5e5e` for secondary text.
- Mute: `#afafaf` for captions and low-priority metadata.
- On dark: `#ffffff` for text on black surfaces.

## Typography

Use a geometric sans. The original reference uses proprietary Uber faces, so this app uses Inter/system fallbacks.

- Display headlines: weight `700`, line-height around `1.1` to `1.2`.
- Body text: weights `400` and `500`, line-height around `1.45` to `1.6`.
- No letter-spacing flourishes.
- No all-caps display headlines.

## Shape and layout

- Pill: `999px`.
- Card: `16px`.
- Small field radius: `8px`.
- A4 PDF page size.
- Desktop web preview centres content in a max-width container.
- PDF pages use full-bleed black or white backgrounds with generous margins.
